Privacy

Questo sito è ospitato su GitHub Pages. I prodotti vengono letti da GitHub Issues pubbliche del repository. La modalità Admin e l’eventuale token GitHub sono salvati solo nel tuo browser (localStorage) e non vengono inviati a terzi (se non alle API di GitHub quando pubblichi un prodotto).

Se condividi il link, gli altri potranno solo visualizzare i prodotti. Non possono pubblicare nulla senza avere un token con permessi sul tuo repository.
